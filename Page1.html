Project Description

Guardian Angel is a secure Home Monitoring Application which utilizes a custom designed circuit. The circuit contains BME280 Sensor and the PIR Motion Sensor. BME280 delivers statistics such as Temperature, Humidity , Pressure. PIR sensor detects motion.
